# TFM-Mortalidad-ENM-PM25

Scripts de R utilizados para el desarrollo del TFM "Mortalidad debida a Enfermedades de las Neuronas Motoras y su posible relación con Partículas Contaminantes PM2.5 en España. Un estudio basado en Sistemas de Información Geográfica" del Máster en Tecnologías de Información Geográfica (TIG).

El orden de ejecución viene marcado al final del nombre del script (por ejemplo, `Limpieza datos población_1` es el 1º). No obstante, a continuación se detalla el orden de ejecución recomendado:

1. Limpieza datos población_1
2. Tratamiento datos motoneurona_2
3. Union_datos_pobl_mort_3
4. Tratamiento municipios segregados_union_pm25_4
5. Municipios_fuera_INE_4_1
6. Matriz adyacencia_5
7. Modificaciones_finales_6
8. Modelo_estadistico_7
